The craft project - most interesting of my works.
Represents an operating system implemented on front-end. See the sources in src/ directory.

Originated as an experimental project driven by a research interest in implementing an operating system based on the state machine concept and the reactive programming paradigm, while strictly adhering to a component-based approach and the layer segregation used in web dev: logic, presentation, data.

The system includes state dispatcher (acting as a kernel), file system, and GUI.

The kernel transforms programmatic and user actions + current state data into input for the next state of components.

The file system is implemented as a tree-like structure with extended functionality, built around the philosophy that “everything is a data node”. A node can be associated with content of any type, will it be text, or remote resource, or another node.

The GUI is inspired by MS Windows and includes desktop, taskbar, navigator, and window system. It provides the following capabilities:

  • Displaying the file system structure in the navigator
  • Displaying shortcuts to selected data nodes on the desktop
  • Rendering data nodes within windows
  • Opening windows via desktop shortcuts, in-window content shortcuts, navigation links, or API calls
  • Equipping windows with functionality such as: open, close, focus, move, resize, switch to preset state (fullscreen, float, full-left, full-right, collapsed)
  • Control window state via title bar controls, taskbar icons, and by snapping windows to screen edges (left, right, top, bottom)
  • Moving windows by dragging the title bar
  • Resizing windows by dragging window borders
  • Maximizing/restoring windows via double-click on the title bar
  • Collapsing/restoring all open windows via a taskbar action

Display and behaviour parameters are fully configurable.

The project is under continuous development as time allow and new ideas income.

The practical example of the system in use is my online CV that is completely implemented on this engine.
